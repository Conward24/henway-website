# Site images (served by Vercel – reliable loading)

Put these image files in this folder with these exact names:

| File name       | What it is                    | Where to get it |
|----------------|-------------------------------|-----------------|
| `logo-black.png` | Henway logo (nav + footer)    | Download from your flask-chatbot repo or use your local copy. |
| `egg-icon.png`   | Egg icon (process cards)      | Same source – the “egg” asset from flask-chatbot. |
| `egg-circuit.png`| Egg/circuit (contact section) | Same as egg icon, or a separate asset if you have one. |
| `mike.jpg`       | Mike’s photo (founder section)| Your chosen photo of Mike; use .png if you prefer. |

After adding the files, commit and push. Vercel will serve them from your domain so they load consistently.
